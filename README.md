# parent-coaching-service-project

초·중·고등학생 자녀를 둔 학부모 대상 진로·학습 코칭 웹서비스 기획 프로젝트입니다.

## 프로젝트 목적

학부모가 자녀의 진로, 학습, 흥미·강점 및 학교생활과 관련하여 가지는 고민을 탐색하고, 이를 바탕으로 학부모가 자신의 고민 상황을 단계적으로 확인할 수 있는 웹 기반 질문 구조를 설계하는 것을 목표로 합니다.

## 주요 기획 내용

- 학부모 고민 유형 탐색
- 진로·학습·강점·학교생활 관련 고민 범주화
- 학부모의 실제 고민 표현 분석
- 초등·중등·고등 학교급별 고민 특성 탐색
- 학부모 대상 Decision Tree 질문 구조 설계

## 데이터 활용

공개적으로 제공되는 데이터 및 Open API를 활용하여 서비스 기획을 위한 탐색적 분석을 진행합니다.

수집된 데이터는 고민 유형과 표현을 분석하고 서비스 질문 구조를 설계하기 위한 참고 자료로 활용합니다.

개인을 식별하기 위한 목적으로 데이터를 활용하지 않습니다.

## Project Status
